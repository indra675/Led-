# 💡 LED Control Systems – 8051 Microcontroller & Arduino UNO

A collection of two embedded systems projects demonstrating LED control using both **8051 Assembly** and **Arduino C/C++**. This project highlights the difference between low-level register programming and high-level embedded development.

---

## 📌 Overview

This repository contains:

- 🔹 8051 LED Rotation using Assembly Language
- 🔹 Arduino UNO LED Chase Effect using C/C++

The projects demonstrate GPIO control, software timing, circuit simulation, and embedded programming concepts.

---

## 🚀 Project 1: 8051 LED Rotation

### Features
- LED rotation on Port 1 and Port 2
- Written in pure 8051 Assembly
- Software delay using nested loops
- Simulated in Proteus
- Developed in Keil uVision5

### Tools Used
- Keil uVision5
- Proteus 8 Professional
- AT89C51 Microcontroller

### Concepts Covered
- GPIO Programming
- RR/RL Instructions
- ACALL & RET
- DJNZ Delay Loops
- Register-Level Programming

---

## 🚀 Project 2: Arduino LED Chase Effect

### Features
- Sequential LED Chase Pattern
- Array-based pin management
- Arduino C/C++
- Simulated in Tinkercad

### Hardware
- Arduino UNO
- 5 LEDs
- 220Ω Resistors
- Breadboard

### Concepts Covered
- Arrays
- Loops
- digitalWrite()
- pinMode()
- delay()

---

## 📊 Comparison

| 8051 | Arduino |
|------|----------|
| Assembly Language | C/C++ |
| Register-Level Programming | High-Level Functions |

| Keil + Proteus | Arduino IDE + Tinkercad |
